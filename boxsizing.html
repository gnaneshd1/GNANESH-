Content Box

Content Box is the default value of box-sizing property. It means that the width and height of the element are calculated including only the content, but not the padding,

THISS BOX usesbox-sizing:content-box.

Border Box

Border Box is the value of box-sizing property. It means that the width and height of the element are calculated including the content, padding, and border, but not the margin.

THISS BOX usesbox-sizing:border-box.
